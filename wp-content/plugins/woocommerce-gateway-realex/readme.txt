=== WooCommerce Realex Gateway ===
Author: skyverge
Tags: woocommerce
Requires at least: 4.1
Tested up to: 4.8.3

Adds the Realex Remote Gateway to your WooCommerce website. Requires an SSL certificate.

See http://docs.woothemes.com/document/realex-payment-gateway/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-realex' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
